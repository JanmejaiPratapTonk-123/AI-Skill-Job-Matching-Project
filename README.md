# AI Skill & Job Matching Platform

An AI-powered platform that helps users discover career paths by analyzing their skills, identifying skill gaps, recommending courses, and matching them with relevant jobs.

## Features

- Resume Skill Extraction (NLP)
- Skill Gap Analysis
- Career Recommendations
- Course Recommendations
- Job Matching
- Government/NGO analytics dashboards
